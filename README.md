# NASA APOD Viewer 🚀

Uma aplicação web construída **React** e **Vite** que consome a API ** Astronomy Picture of the Day(APOD) ** da NASAç. O projeto exibe diariamente uma imagem (ou vídeo) do universo, acompanhado de titulo, descrição e outras infromações.
<img src![screenshot ](image.png)>

## Tecnologias Utilizadas

-React
-Vite
-Javascript(ES6+)
-CSS
-API publica da NASA (APOD)

##Funcionalidades
-Exibição de imagens e vídeos do dia
-Titulo, descrição e data
-Indicador de carregamento

## 🚀Deploy

A aplicação está disponivel atraves da plataforma Vercel.
